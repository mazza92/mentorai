# Lurnia

Your AI Learning Companion for YouTube

## Overview

Lurnia transforms YouTube content into searchable knowledge. Ask questions about any video or channel and get instant answers with timestamp citations.

## Features

- **Channel Import**: Import entire YouTube channels for comprehensive Q&A
- **Video Q&A**: Ask questions about specific videos with timestamp references
- **Chrome Extension**: Seamless integration directly on YouTube
- **Multi-language Support**: English and French interfaces

## Tech Stack

- **Frontend**: Next.js 14, React, Tailwind CSS, TypeScript
- **Backend**: Node.js, Express
- **Database**: Google Firestore
- **AI**: Google Gemini API
- **Auth**: Supabase
- **Payments**: Stripe

## Development

```bash
# Install dependencies
npm run install:all

# Run development servers
npm run dev
```
